其中有代码风格不统一，复用率低，不规范等问题，这是因为1.我想多试试几种不同的方法；2.因为是小项目，想省点时间去复习，所以有时候就粘贴了，提高复用率以后会多研究；3.规范慢慢改进吧。这里只是给自己一个记录和问题总结，当然欢迎交流。
